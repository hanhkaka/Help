1) install NodeJs
2) install following modules;

	npm install mysql cheerio connect express foscam node-schedule node-speak node-tts-api nodemailer nodetime passport passport-local request sequelize sequelize-mysql socket.io websocket.io ws xml-digester xml2js xmlbuilder xml xmldom

3) Create table example:

	var Device = sequelize.define('nimbus_device', {

		nID : {
			type : NIMBUS_DB.STRING,
			primaryKey : true
		},
		driver : NIMBUS_DB.STRING
	});
	
	
	Note: Table name should in with plural forms (nimbus_devices) and use table  name in define with singular forms ;
	
4) Debug error with following command:

	process.on('uncaughtException', function(err) { console.log(err); });
